# SueLr-ComfyUI

一个基于 Vite + React + Node.js 的可视化流程编排工具。

---

## 📦 项目结构

* `src/`：前端代码
* `backend/`：后端服务

---

## 🚀 快速开始

### 安装依赖

```bash
npm install
cd backend
npm install
cd ..
```

---

### 配置环境变量

复制：

```bash
cp .env.example .env
```

然后填写你的 API Key 等信息。

---

### 启动项目

前端：

```bash
npm run dev
```

后端：

```bash
cd backend
npm start
```

---

## 🔒 注意

* `.env` 不会上传（保护隐私）
* 只提供 `.env.example` 模板
